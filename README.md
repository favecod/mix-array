## How to use?
For example, we first define an array.
```js
 let array = [1,2,3,4,5,6]
 ```
 Then we create a new class of mix and pass the array to it.
 ```js
 new Mix(array)
 ```
Mixed array is ready for use!
```js
[6,3,5,1,4,2]
```
